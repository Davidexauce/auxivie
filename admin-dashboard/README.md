# Admin Dashboard Next.js Project

## Overview
This is the admin dashboard frontend project built with Next.js. It serves as the UI for managing users, documents, reservations, payments, reviews, disputes, and platform settings.

## Project Structure
- `pages/` : Next.js pages directory
- `components/` : Reusable UI components
- `styles/` : CSS or styling files

## Getting Started
1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

3. Build for production:
```bash
npm run build
npm start
```
